Metodologías más usuales para la seguridad informática

Las metodologías más comunes para la seguridad informática incluyen:

  • ISSAF: Information Systems Security Assessment Framework.
  • OWASP: Open Web Application Security Project.
  • OSSTMM: Open Source Security Testing Methodology Manual.
  • ISO/IEC 27000: Familia de estándares para la gestión de la seguridad de la información.
